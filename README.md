# BioPlant Aquatic Habitat

BioPlant Aquatic Habitat is a **speculative biomedical and biohybrid research repository** exploring whether future gas-exchange interfaces, oxygen-carrying nanomaterials, and physiological control systems could support extended aquatic survival and work.

## Current status
**Concept stage / preclinical framing only**

This repository does **not** describe a validated medical product. It is an early research and venture-framing project intended to:
- organize the scientific concept,
- identify validation gaps,
- structure preclinical milestones,
- support credible investor and research discussion.

See [DISCLAIMER.md](DISCLAIMER.md).

## Core concept tracks
The project is organized into three research tracks:

### 1. Bio-Interface Gas Exchange
Explores biomimetic oxygen extraction and carbon dioxide handling interfaces inspired by aquatic respiration.

### 2. Oxygen-Carrying Nanomaterials
Explores respirocyte-like or perfluorocarbon-based oxygen transport concepts for temporary physiological support.

### 3. Physiological Sensing and Control
Explores sensing, feedback, and safety logic for oxygen, carbon dioxide, pH, thermal stress, and depth-related constraints.

## Why this repository exists
Human survival in aquatic environments is limited by:
- insufficient oxygen access,
- carbon dioxide accumulation,
- pressure tolerance,
- thermal stress,
- lack of adaptive physiological control.

This repository frames a long-horizon biomedical research question:

> Can future biohybrid systems safely augment human aquatic endurance through gas-exchange support, nanomedical oxygen transport, and closed-loop physiological monitoring?

## Validation position
This concept is **not validated** for human use.

To become a biomedical product, the project would need evidence across:
1. scientific plausibility,
2. bench validation,
3. material biocompatibility,
4. preclinical animal testing,
5. clinical testing,
6. regulatory review.

## First executable milestone
The first real milestone is **not human implantation**.

It is a controlled benchtop program to test whether a prototype oxygen-transfer membrane system can measurably improve dissolved oxygen transfer under physiological boundary conditions.

See `docs/experimental-milestone-1.md`.
